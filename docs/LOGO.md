# Logo y marca VeloCity

## Uso del nombre
- **VeloCity** (una palabra, C mayúscula).
- Eslogan: **Tu ciudad en movimiento.**

## Colores oficiales (paleta empresarial)
| Nombre    | Hex       | Uso principal      |
|-----------|-----------|--------------------|
| Primario  | `#F46E20` | Naranja: botones, links, acentos, logo |
| Secundario| `#3F474A` | Gris carbón: texto, fondos corporativos |
| Oscuro    | `#3F474A` | Cabeceras, cuerpo de texto |
| Claro     | `#F8FAFC` | Fondos claros |
| Acento    | `#5BA4D4` | Azul: líneas geométricas en uniformes/casco |

## Concepto del logo
- **Símbolo:** Combinación de “velocidad” (líneas dinámicas o rayo) con referencia a ciudad/movilidad (rueda o silueta urbana). Minimalista, legible en tamaños pequeños (favicon, app icon).
- **Tipografía:** Sans-serif moderna (ej. Outfit, Space Grotesk). La “V” puede usarse como icono solo (app, favicon).
- **Formato:** SVG para web y exportaciones en 512×512 y 1024×1024 para app stores.

- `apps/web/public/favicon.ico` — generado desde el símbolo.
